# To-Do-App
Project Description: A personal task tracker to mitigate task tracking and improve personal productivity.

## Tools/Frameworks: 
- HTML<br/>
- CSS<br/>
- JavaScript<br/>
- Cloudflare, Font Awesome libraries

## Development Issues/Complications and Solutions: 
	- Adding goals to the list<br/>
	- Progress bar 
	  - # instead of .
	- Circle altering shape as progress ensues<br/>
	- Empty image disappearing and reappearing when task(s) is entered 
	  - (task-list width from 100% to 300%)

## 3 Major Takeaways:
	1. Debugging is an essential skill that will save both time and resources.<br/>
	2. Consistency + Controlled breaks = Steady progress without burnout.<br/>
  	3. On future projects, create documentation to reduce time with bug search.
