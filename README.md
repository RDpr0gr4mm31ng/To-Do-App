# To-Do-App
Project Description: A personal task tracker to mitigate task memorization and improve personal productivity.

Tools/Frameworks: HTML, CSS, and JavaScript with Cloudflare and Font Awesome libraries

Development Issues/Complications and Solutions: 
	• Adding goals to the list
	• Progress bar (# instead of .)
	• Circle altering shape as progress ensues
	• Empty image disappearing and reappearing when task(s) is entered

3 Major Takeaways:
	1. Debugging is an essential skill that will save both time and resources. 
	2. Consistency + Controlled breaks = Steady progress without burnout.
  3. On future projects, create documentation to reduce time with bug search.
